OpenGeo Suite QGIS plugin
=========================

The GeoServer QGIS plugin is used to configure GeoServer through QGIS. It allows you to easily configure your GeoServer instance, from preparing data and styling to publishing directly to a catalog, all through the QGIS interface. This manual describes how to use the GeoServer QGIS plugin and the operations that it supports .
